# CoHealth

A simple interactive web application built using **Flask** and **SQLite**.  
This project was created as part of the Web Applications course.

## Features

- User Registration (Sign Up)
- User Login (Authentication)
- Home Page after Login
- Services Page
- Online Health Checkup Form
- 24/7 Support Page
- Health Records Page
- All pages built using HTML + TailwindCSS

## Technologies Used
- Python (Flask)
- SQLite (Database)
- HTML / Tailwind CSS
- Jinja2 Templates
