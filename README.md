CoHealth – Web Application

A fully functional interactive web application built using Flask, SQLite, and Jinja2 Templates.
This project was developed as part of the Web Applications course requirements.

 Features

User Registration (Sign Up):
Allows new users to create an account and store their credentials securely in the database.

User Login (Authentication):
Validates user credentials and grants access to the system.

Home Dashboard:
A personalized home page displayed after successful login.

Services Page:
A section containing links to all available healthcare services.

Online Health Checkup Form:
Users can submit symptoms and receive a basic automated evaluation.

24/7 Support Page:
A form where users can contact support at any time.

Health Records Page:
A simple page representing the user's health information.

UI built with HTML + TailwindCSS:
Clean, modern, and responsive user interface.

 Technologies Used

Python (Flask Framework) – Backend logic, routing, server-side processing

SQLite – Lightweight relational database

HTML + TailwindCSS – Front-end structure and styling

Jinja2 – Template engine used inside Flask
